安装ArchLinux教程

发表于 2020-08-24 更新于 2020-08-25 分类于 笔记
本文字数： 1.2k 阅读时长 ≈ 1 分钟

安装前的准备

  1. 首先前往 >> 清华大学开源镜像站 下载 Arch Linux 镜像，制作U盘启动盘。
  2. 开机进入U盘启动(UEFI引导)，进入系统后将会以 root 身份登陆进一个虚拟控制台，默认的 shell 是 zsh
  3. 进行联网
    • 确保系统已经启用了网络接口，用 ip link 命令检查
    • 要连接到网络，可以：
      • 有线以太网–连接网线
      • WiFi–使用 iwctl 验证无线网络
    • 配置网络连接
      • DHCP：动态 IP 地址和 DNS 服务器分配
      • 静态 IP 地址
    • 用 ping 命令检查网络连接
  4. 更新系统时间
  5. 建立硬盘分区，使用 fdisk 或 cfdisk 工具进行硬盘分区
    挂载点分区分区类型建议大小
    /mnt/boot/EFI/dev/sdx1EFI 系统分区260-512MB
    /mnt//dev/sdx2Linux x86_64 根目录剩余空间
    [SWAP]/dev/sdx3Linux swap大于 512MB
  6. 格式化分区
  7. 挂载分区

安装

  1. 选择镜像，编辑 /etc/pacman.d/mirrorlist 文件，选择镜像源
  2. 安装必须的软件包

配置系统

  1. 生成 fstab 分区表
  2. 进入新安装的系统
  3. 设置时区
  4. 本地化
  5. 安装上网的软件
  6. 使用 passwd 命令设置 root 用户密码
  7. 安装引导并配置
  8. 退出并重启
